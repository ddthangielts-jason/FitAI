Hướng dẫn Deploy & Ra mắt

Đọc từ trên xuống, làm theo thứ tự. Tổng thời gian: 2–3 tiếng.

1
Chuẩn bị thư mục upload
5 phút
01
MIỄN PHÍ

Kiểm tra thư mục fitai-deploy/

Thư mục bạn download từ mình phải có đúng cấu trúc này:

fitai-deploy/
├── index.html     ← Landing page
├── app.html       ← Ứng dụng chính
├── exercises.html ← Thư viện bài tập
├── vercel.json    ← Cấu hình Vercel
├── .env.example  ← Mẫu env variables
└── api/
    ├── chat.js     ← AI proxy
    ├── checkout.js ← Stripe payment
    └── webhook.js  ← Stripe webhook

Nếu thiếu file nào, nhắn mình build lại.

2
Deploy lên Vercel
15 phút — miễn phí
02
MIỄN PHÍ

Tạo tài khoản Vercel

Mở trình duyệt, vào địa chỉ:

https://vercel.com/signup

Nhấn "Continue with Google" → đăng nhập Google của bạn → xong.

03

Upload thư mục lên Vercel

Trong Vercel dashboard:

1. Nhấn nút "Add New..." → chọn "Project"

2. Kéo và thả toàn bộ thư mục fitai-deploy/ vào ô upload

3. Đặt tên project: fitai

4. Nhấn "Deploy" → chờ 1–2 phút

✅ Sau khi deploy xong, Vercel cho bạn URL dạng: fitai-xxx.vercel.app — đây là link app thật của bạn!
04
QUAN TRỌNG

Lấy Anthropic API Key và paste vào Vercel

Bước 4a — Lấy API key:

https://console.anthropic.com/api-keys

Nhấn "Create Key" → copy key (bắt đầu bằng sk-ant-...)

Bước 4b — Paste vào Vercel:

Trong Vercel: Settings → Environment Variables → Add New

Điền vào 2 ô:

NAME: ANTHROPIC_API_KEY
VALUE: sk-ant-xxxxxxxx (key bạn vừa copy)

Nhấn Save → sau đó nhấn Redeploy để apply.

⚠️ Tại sao cần làm bước này? Nếu không làm, API key bị lộ trong code — ai cũng thấy và dùng được, tốn tiền của bạn.
3
Tích hợp Stripe — nhận tiền
45 phút
05
CẦN ĐĂNG KÝ

Tạo tài khoản Stripe

https://stripe.com → Create account

Điền thông tin:

  • Email của bạn
  • Tên thật + số CMND/CCCD
  • Địa chỉ tại Việt Nam
  • Tài khoản ngân hàng VN để nhận tiền
⏳ Stripe duyệt trong 1–3 ngày làm việc. Bạn vẫn có thể test bằng tài khoản test mode ngay.
06

Tạo sản phẩm FitAI Pro trên Stripe

Trong Stripe Dashboard:

1. Vào Products → Add product

2. Tên sản phẩm: FitAI Pro

3. Nhấn "Add price" → chọn Recurring

4. Tạo Price đầu tiên:

Currency: VND
Amount: 1499000
Billing period: Yearly
→ Copy Price ID: price_xxxxxxxxxx (LƯU LẠI)

5. Nhấn "Add another price" → tạo Price thứ 2:

Currency: VND
Amount: 249000
Billing period: Monthly
→ Copy Price ID: price_xxxxxxxxxx (LƯU LẠI)
07

Paste Stripe keys vào Vercel

Vào Vercel → Settings → Environment Variables, thêm 4 biến:

STRIPE_SECRET_KEY = sk_live_xxxxxxxx
    (Stripe → Developers → API Keys → Secret key)

STRIPE_PRICE_ANNUAL = price_xxxxxxxx
    (Price ID năm bạn vừa tạo)

STRIPE_PRICE_MONTHLY = price_xxxxxxxx
    (Price ID tháng bạn vừa tạo)

NEXT_PUBLIC_BASE_URL = https://fitai-xxx.vercel.app
    (URL Vercel của bạn)

Sau khi điền xong → nhấn Redeploy.

08

Cấu hình Stripe Webhook

Webhook giúp app biết khi user đã trả tiền thành công.

Trong Stripe: Developers → Webhooks → Add endpoint

Endpoint URL:
https://fitai-xxx.vercel.app/api/webhook

Events to listen:
✓ checkout.session.completed
✓ customer.subscription.deleted
✓ invoice.payment_failed

Sau khi tạo → copy Signing secret (whsec_xxx) → paste vào Vercel:

STRIPE_WEBHOOK_SECRET = whsec_xxxxxxxx
4
Test toàn bộ trước khi ra mắt
20 phút
09

Test Stripe bằng thẻ test

Stripe cung cấp số thẻ test để bạn thử thanh toán mà không mất tiền thật:

Số thẻ: 4242 4242 4242 4242
Ngày hết hạn: bất kỳ (VD: 12/28)
CVC: bất kỳ (VD: 123)
Tên: Test User

Mở app → nhấn Nâng cấp Pro → điền thẻ test → kiểm tra xem redirect về app không.

✅ Nếu redirect thành công về app.html?pro=true → Stripe hoạt động đúng!
10

Test AI Coach

Vào tab AI Coach → hỏi một câu → đợi trả lời.

✅ Nếu AI trả lời tiếng Việt → API key hoạt động đúng!
Nếu lỗi "API key chưa cấu hình": Quay lại Vercel → kiểm tra ANTHROPIC_API_KEY đã được save và Redeploy chưa.
